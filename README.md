
# Tic Tac Toe

### LVL1:

    <---------------Includes--------------->

    Player vs Player
    Simple Two Player console Code 

### LVL2:

    <---------------Includes--------------->

    P: PC vs PC
    Y: PC vs Human
    N: Human vs Human
    Q: Quit
    Uses Randomisation ---> __PC is BRAINLESS__

### LVL3:

    <---------------Includes--------------->

    Never win!
    PC will Defend Every Move You Play
    PC may not rush to win but won't let you win
